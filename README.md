 # SOC Detection Engine

Simulação de um ambiente SOC para detecção de incidentes de segurança via análise de logs.

## Funcionalidades

- SSH Brute Force Detection (MITRE T1110)
- Port Scan Detection - Nmap (MITRE T1046)
- Blacklist Detection
- Classificação de severidade (LOW, MEDIUM, HIGH)
- Geração de relatório automático
- Registro de incidentes em JSON

## Como executar

python analyzer.py
